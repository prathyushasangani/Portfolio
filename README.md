# Prathyusha Sangani Portfolio

Static personal website for `prathyushasangani.com`.

## Files

- `index.html` - page content
- `styles.css` - visual design and responsive layout
- `script.js` - current year helper
- `CNAME` - custom domain for GitHub Pages

## Publish Options

Upload these files to any static host such as GitHub Pages, Netlify, Vercel, Hostinger, or cPanel public HTML.

For GitHub Pages:

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Enable Pages from the repository settings.
4. Set the custom domain to `prathyushasangani.com`.
5. In your domain DNS, point the domain to the hosting provider.
